# Prototipo web de orientación vocacional

Este proyecto corresponde al desarrollo de un prototipo web de orientación vocacional basado en Machine Learning para apoyar la recomendación académica por áreas en estudiantes de bachillerato de Guayaquil.

## Estructura inicial del proyecto

- `frontend/`: interfaz web del estudiante y administrador.
- `backend/`: API, autenticación, procesamiento de respuestas e integración futura con modelos de Machine Learning.
- `docs/`: documentación técnica, mockups, diagramas y evidencias del desarrollo.

## Tecnologías previstas

- Frontend: React, TypeScript y Tailwind CSS.
- Backend: FastAPI con Python.
- Base de datos: PostgreSQL.
- Machine Learning: XGBoost y MLP.
- Entorno de desarrollo: Visual Studio Code.
